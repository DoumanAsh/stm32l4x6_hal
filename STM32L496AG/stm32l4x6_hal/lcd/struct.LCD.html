pub fn new(lcd: LCD, config: Config) -> Self
[src]

Initializes LCD

Requirements

  • Pre-initialize HW using init method.
  • Configure LCD GPIO pins as alternative functions. You should check board's documentation for pins.
  • Verify configuration through validate function

Steps:

  1. Turns off.
  2. Reset RAM registers and set update request.
  3. Performs configuration.
  4. Turns on.
